## Instalación
***

1. Open the folder where you want to copy the repository with your IDE.
2. Clone the repository:
    ```bash
    git clone git@github.com:RafaelOrti/app-ia-peliculas.git
    ```
3. Access the folder:
    ```bash
    cd app-ia-peliculas
    ```
4. Install the dependencies:
    ```bash
    npm i
    ```
5. Initialize the server:
    ```bash
    npm start
    ```
